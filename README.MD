##  Overview
This web-based interactive mapping application allows users to measure distances and calculate areas effortlessly. Built with **Leaflet.js** and **Leaflet Draw**, the tool supports:
- **Geolocation**
- **Location search**
- **Multiple map layers**
- **GeoJSON export**

Perfect for **GIS professionals**, **students**, and **mapping enthusiasts**!

##  Key Features
‚úÖ **Distance Measurement**: Click on the map to add points and calculate distances in **kilometers, miles, or meters**.

‚úÖ **Area Calculation**: Draw polygons and instantly get their area in **square kilometers**.

‚úÖ **Location Search**: Use OpenStreetMap‚Äôs **Nominatim API** to find locations worldwide.

‚úÖ **Export as GeoJSON**: Save drawn features in **GeoJSON** format for GIS applications.
